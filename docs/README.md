# GoPUP概况

- 本文档更新于 2020-10-28;
- 如有库或者文档的相关问题, 请在 [GoPUP Issues](https://github.com/justinzm/gopup/issues) 中提 Issues;

风险提示：GoPUP 项目所采集的数据皆来自公开的数据源，不涉及任何个人隐私数据和非公开数据。 同时本项目提供的数据接口及相关数据仅用于学术研究，任何个人、机构及团体使用本项目的数据接口及相关数据请注意商业风险。
